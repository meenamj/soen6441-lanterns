Instructions for how to compile, build and run your code

1. Extract the zip file of the project.
2. Open the same in your Java Workspace.
3. Open Eclipse and Open the project in it.
4. When completed run the Game class.

---------------------------------------------------------------------------------------
To display Unicode symbols on the console

-	Go to Windows -> preferences -> General -> Workspace
-	Change text file encoding to UTF-8
-	Apply changes
-	Click OK

---------------------------------------------------------------------------------------
Eclipse plugin ANSI Escape in Console lugin is used to display colors in the console

How to install the ANSI escape in console

- To install the plugins in Eclipse select “Help” -> “Install New Software…” and click “Add…” to add the following URL:

- http://www.mihai-nita.net/eclipse
